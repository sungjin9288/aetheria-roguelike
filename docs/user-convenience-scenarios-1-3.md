# 사용자 편의 기능 시나리오 1~3

## 시나리오 1: 빠른 시작 가이드 (Quick Start Guide)
- 목적: 신규 유저가 첫 5분 내 핵심 루프(이동 -> 탐색 -> 전투 -> 보상)를 이해하도록 지원
- 트리거: 캐릭터명 입력 후 첫 접속
- 사용자 흐름:
1. `시작의 마을`에서 하단에 "추천 행동 3단계" 카드 노출
2. `MOVE`, `EXPLORE`, `REST` 버튼에 순서 배지 표시
3. 단계 완료 시 체크 표시와 다음 단계 자동 하이라이트
- 기대 효과: 초반 이탈률 감소, 튜토리얼 텍스트 의존도 축소

## 시나리오 2: 인벤토리 스마트 정리 (Smart Inventory)
- 목적: 장비/소모품 탐색 시간을 줄여 전투 진입 속도 개선
- 트리거: 인벤토리 탭 진입 시
- 사용자 흐름:
1. 상단에 `장비`, `소모품`, `재료` 빠른 필터 제공
2. 장비 아이템 선택 시 현재 장착 대비 `ATK/DEF 증감`을 즉시 표시
3. `추천 장착` 버튼으로 직업/티어/능력치 기준 최적 장비 1클릭 적용
- 기대 효과: 장비 교체 실수 감소, 전투 준비 시간 단축

## 시나리오 3: 전투 후 자동 정리 (Post-Combat Assistant)
- 목적: 반복 플레이 구간의 피로도를 낮추고 의사결정 부담 완화
- 트리거: 전투 종료 직후 (승리/패배 복귀 포함)
- 사용자 흐름:
1. 전투 결과 요약 카드: 획득 골드/경험치/핵심 루트 자동 요약
2. HP/MP가 임계값 미만이면 `휴식` 또는 `회복 아이템 사용` 추천 액션 표시
3. 인벤토리 과밀 시 `가치 낮은 재료 일괄 판매` 제안(확인 후 실행)
- 기대 효과: 반복 입력 감소, 플레이 템포 유지, 편의성 체감 상승
